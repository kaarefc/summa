This package defines a simple ContentHandler and Parser mechanism. It can be used to ingest any XML, where a simple mapping between the XML source and the two key entities: recordID and record boundaries exists. NO namespace can be given record. To use this properly: identify the "Record tag" and the ID. tag. From the ingest configuration choose the SimpleDirectoryDigester: 
{@code
   /home/summa/data/danskReklameFilm
   false
   dk.statsbiblioteket.summa.ingest.SimpleXML.SimpleDirectoryDigester
   false
   UTF-8
   
   true
   id
   mdcs
   reklamefilm
}
