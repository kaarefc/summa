Provides for fast remote communication, meant for simple request-response. The Fast framework is a very light weight communication framework with the exchange protocol largely defined by implementations. The main purpose of the framework is to handle network failures and reconnections.

The Fast framework is used by several components in Summa: - Ingest: Record traffic - Index: TermVector exchange - Search: Distributed search - Facet Browsing: Distributed browsing - Cluster: Centroid-building
